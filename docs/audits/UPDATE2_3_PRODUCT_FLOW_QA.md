# Update 2.3 Product Flow QA

## Command Flow

Input:

`Create a 30-second Facebook reel for UK stable owners.`

Expected inference:

- Platform: Facebook
- Format: reel
- Duration: 30 seconds
- Audience: UK stable owners
- Goal: stable owners/signups
- Needs video/script/caption/media plan

## Studio Output

When AI is ready, the generated draft appears immediately in the Studio panel and includes:

- strategy
- hook/content preview
- shot list
- script/body
- caption
- CTA
- hashtags
- visual direction
- media plan
- recommended schedule
- compliance notes
- growth score

## Actions

- Edit
- Regenerate
- Improve hook
- Make shorter
- Make more premium
- Generate image/video when media is ready
- Send to approval
- Schedule

## Audience And Suppression

- Audience add/search/list/export remains wired to existing admin marketing contact procedures.
- Suppression add/remove/list remains wired to existing unsubscribe/contact logic.
- No send-path suppression behavior was loosened.

## Assets

- Assets tab shows real generated media when present.
- Prompt-only, missing provider, and failed states are visible and truthful.
- No fake image/video success is introduced.
